Welcome to abu.admin's documentation!
=====================================

简介
----

abu.admin 致力于简化 Python 编写的应用程序部署，特别是需要在同一台物理机器上部署多个项目运行环境的情况下尤其有用。它提供简洁的命令行接口方便用户使用。理论上来说，任务项目都可以使用 abu.admin 进行部署，只要该项目实现 abu.admin 的接口协议。

abu.admin 受到 TracAdmin 的启发，在此表示感谢。

安装
----

abu.admin 已经发布到 pypi，所以通过执行 easy_install/pip 即可完成安装 ::
    easy_install -U abu.admin
安装时，可能需要超级用户权限或者指定 -i 参数。
